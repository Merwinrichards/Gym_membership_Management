   - Change port in application.properties

   - Run: `.\mvnw clean install`

4. **JS Not Working**
   - Clear cache: `Ctrl+Shift+Delete`
   - Hard refresh: `Ctrl+Shift+R`

   - Check browser console (F12)
   - Verify API response
